## 🚀 部署流程

### 用户需要做的步骤：
1. **下载ZIP包** - 解压到本地
2. **上传到阿里云** - 通过网页界面上传
3. **安装依赖** - 运行一条命令
4. **测试运行** - 运行测试脚本
5. **设置定时** - 配置Cron任务

### 技术细节：
- **Python版本**: 3.7+
- **核心依赖**: feedparser 6.0.10
- **API接口**: arXiv官方API (http://export.arxiv.org/api/query)
- **数据格式**: Atom XML
- **输出格式**: Markdown

## 🔧 配置文件详解

### config.yaml 默认配置：
```yaml
# 搜索配置
search:
  keywords:
    - "quantum spin liquid"      # 量子自旋液体
    - "magnetoelectric coupling" # 磁电耦合
    - "multiferroic"             # 多铁性
    - "topological insulator"    # 拓扑绝缘体
    - "skyrmion"                 # 斯格明子
    - "spintronics"              # 自旋电子学
  
  days: 7           # 搜索最近7天
  max_results: 10   # 每关键词最多10篇
  
# 输出配置
output:
  format: "markdown"
  save_to_file: true
  report_dir: "./reports"
  
# 定时任务配置
cron:
  enable: true
  schedule: "0 9 * * *"  # 每天9点
  timezone: "Asia/Shanghai"
```
